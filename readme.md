# 🏛️ Monitor Legislativo — Senado Nacional

Monitor automático de los **72 senadores** de la Nación Argentina.

## 📊 KPIs
- Participation Index (votos emitidos)
- Distribución por Bloque político
- Reporte por Provincia
- Votos Afirmativos / Negativos / Abstenciones

## 🚀 Uso local
pip install -r requirements.txt
python scraper_senadores.py

## 🤖 GitHub Actions
Corre automáticamente cada día a las 08:00 (ARG) y guarda CSV en data/
